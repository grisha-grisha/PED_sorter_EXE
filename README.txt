"Обработчик пакетов сметной документации" - это десктопная программа, предназначенная для упрощения процесса типизации и переименования файлов пакетов сметной документации в соответствии с документом "Требования к формированию пакета сметной документации 04.08.2025". Разработал Силантьев Григорий для "ГСП Ремонт" август 2025г.

Инструкция по развороту для пользователя:
Скачать zip-архив, разархивировать. Запустить PED_Sorter.exe

Инструкция по использованию программы представленна в "MANUAL-PED_SORTER.docx"
