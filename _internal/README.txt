"Обработчик пакетов сметной документации" - это десктопная программа, предназначенная для упрощения процесса типизации и переименования файлов сметной документации в соответствии с документом "Требования к формированию пакета сметной документации 04.08.2025". Разработал Силантьев Григорий для "ГСП_Ремонт" август 2025г.

*инструкция по развороту для разработчиков*
клонировать репозиторий на компьютер. создать виртуальное окружени, установить зависимости командой "pip install -r requirements.txt". запуситить файл "PED_Sorter.py" (в нем находится вся основная логика.)

"PED_design.ui и "tags_window_design.ui" - файлы дизайна, созданы в программе для разработки интерфейса "QT Designer". их можно открыть в ней как проекты.

"PED_design.py и "tags_window_design.py" - файлы дизайна. конвертированны из ui в py с помощью команд в консоли: "pyside6-uic PED_design.ui -o PED_design.py " и "pyside6-uic tags_window_design.ui -o tags_window_design.py", чтобы подключить дизайн к главному рабочему файлу "PED_Sorter.py"

в "file_types_base.json" хранится словарь с перечнем файлов сметной документации и их тэгами.

Tesseract-OCR - папка с программой для распознования текста в PDF файлах
